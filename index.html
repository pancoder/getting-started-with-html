Intro to the Web: Notes

The Web was born in the 1990's. The World Wide Web is made up of HTML documents connected by hyperlinks thus creating a web of documents that are stored on servers all around the world. The servers communicate to each other through a protocal such as HTTP.

What is HTML?

1. The text that you see
2. Mark up language that determines what the page looks like.
3. Reference to other documents
4. References to other pages.

Try it out! Use HTML editor at www.udacity.com/playground

Markup
A markup code is called an Element.
Looks like: (left arrow)name(/right arrow)contents(left arrow)name(/right arrow)

Question: How do I type out the left and right arrow in HTML if I want to see them printed out and not as a part of code?

Self-Answer: You need entities! Reserved characters in HTML must be replaced with character entities.
The left angled bracket entity is: & + l + t + ; (omit the plus signs)
The right angled bracket entity is: & + g + t + ; (omit the plus signs).
Here it is in action: < and >
Characters, not present on your keyboard, can also be replaced by entities.

Tags

Tags are what you type before and after each markup instruction. There is an opening tag and a closing tag. It is a good habbit to create both the opening and closing tag at the same time so you don't forget.
Note: When you are writing code there is no need for spaces between the content and the open or closed tags.

Attributes

Attributes are element modifiers with a "value" that changes the content enclosed in an element.
Attribuites look like: <tag ATTR="value">content <tag>

Creating Links
An Anchor tag is an attribute that is used for making links.

Example: <a href="value">content</a>

Visit Udacity HTML editor! Note: you must include the space between the a and the href.

Question: What is a void tag?
Self-Answer: A void tag is the kind of tag that does not requre a close tag. Most HTML elements require an open and closed tag. Void or Empty tags are exceptions to that rule because they, in themselves, do not have conent. So a Break tag <br> is a void tag.

In-line elements vs. Block elements

HTML elements can be 'In-Line' or 'Block'. Block elements create an invisible box around the content.

Example: <b> ,<em>,<img>,and <br> are inline elements. <p> ,</p>,<div> </div> are block elements
